# Install

```bash
pnpm install
```

# Run

```bash
pnpm dev
```

# Env

add the env to set the request url

```bash
copy ./example.env .env

# then you can set the url in .env
# example: 
#VITE_REQUEST_URL=https://example.com/getJsonData
#VITE_TITLE=Custom Data View
```

# Build

```bash
pnpm build
```
